[문제 링크](https://programmers.co.kr/learn/courses/30/lessons/17677)

2. 추출한 집합에서 `교집합`과 `교집합이 아닌 집합` 원소를 찾는다.
3. (교집합개수/합집합개수)*65536 정수형 값을 반환한다.
