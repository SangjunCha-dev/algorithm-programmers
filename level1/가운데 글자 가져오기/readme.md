## 가운데 글자 가져오기

분류 : 연습문제

[문제 링크](https://programmers.co.kr/learn/courses/30/lessons/12903)

1. 문자열 `s` 길이값 나누기 연산값을 `index`변수에 저장
2. `s` 길이값이 나머지 연산으로 홀수이면 `s[index]` 반환
3. `s` 길이값이 나머지 연산으로 짝수이면 `s[index-1:index+1]` 반환
